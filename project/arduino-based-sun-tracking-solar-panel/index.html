Arduino Based Sun Tracking Solar Panel

Sareh Soltani
Jan 11, 2017
Code

This project was done as the final project of the Interface Circuit Design course, taught by Prof. Mahdi Sedighi, Tehran Polytechnic.
In modern solar tracking systems, the solar panels are fixed on a structure that moves according to the position of the sun. In this project, I implemented a solar tracker system using two servo motors, 4 light sensors and Arduino UNO board to rotate the solar panel towards the Sun or a source of light.
I used a standard servo that can rotate approximately 180 degrees (90° in each direction) and is controlled using the included Arduino’s Servo Library.
LDRs are used as the main light sensors. Two servo motors are fixed to the structure that holds the solar panel. The program for Arduino is uploaded to the microcontroller. The working of the project is as follows:

  • LDRs sense the amount of sunlight falling on them. Four LDRs are divided into top, bottom, left and right.
  • For east–west tracking, the analog values from two top LDRs and two bottom LDRs are compared, and if the top set of LDRs receives more light, the vertical servo will move in that direction. If the bottom LDRs receives more light, the servo moves in that direction. For the angular deflection of the solar panel, the analog values from two left LDRs and two right LDRs are compared. If the left set of LDRs receives more light than the right set, the horizontal servo will move in that direction. If the right set of LDRs receives more light, the servo moves in that direction.
